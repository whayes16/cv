Scoring Machine (Playmaking)

The scoring machine can be built in multiple ways as you will see there is another scoring machine build I have provided. This section will be for the scoring machine with a playmaking emphasis.

    Pro's

  • Great shooting both standing still and off the dribble
  • Great ball handling with above average passing skills
  • Above average back to the basket game especially on smaller guards
  • Very strong isolation player can efficiently create their own shot in tough situations

      • Con's

    • Weak finishing ability especially in traffic
    • Can lack the passing ability of some play making builds which requires us to take some play making badges that we would not normally take on other builds
    • Weak defensivley
